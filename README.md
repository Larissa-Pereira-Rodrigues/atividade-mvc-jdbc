# Atividade MVC + JDBC + PostgreSQL

Projeto desenvolvido para a disciplina de Estruturação de Sistemas.

## Cenários desenvolvidos

### Cenário 1
Clínica Veterinária

### Cenário 2
Oficina Mecânica

### Cenário 3
Escola de Cursos Livres

## Tecnologias

- Java
- Maven
- PostgreSQL
- JDBC
- MVC

## Organização

```
controller
model
repository
service
util
Main.java
```

Cada cenário foi desenvolvido em uma branch própria:

- cenario1
- cenario2
- cenario3

Após a conclusão de cada cenário, a branch foi integrada à branch principal (`main`).
